          SECTION 2: AI FINANCIAL AGENT - DETAILED PLAN
==============================================================================

How the AI Agent Would Work:
----------------------------
The agent analyzes your existing SQLite database for:
- Monthly income vs expenses ratio
- Category-wise spending patterns
- Budget adherence history
- Spending trends (increasing/decreasing)


Types of Suggestions:
---------------------

A) EXPENSE SUGGESTIONS:
   ┌─────────────────────────────┬───────────────────────────────────────────────────────────────────┐
   │ Trigger                     │ Example Suggestion                                                │
   ├─────────────────────────────┼───────────────────────────────────────────────────────────────────┤
   │ Overspending in a category  │ "You spent ₹8,000 on food — 60% more than your 3-month avg.     │
   │                             │  Try meal prepping to save ~₹3,000"                               │
   ├─────────────────────────────┼───────────────────────────────────────────────────────────────────┤
   │ Budget almost exhausted     │ "You've used 85% of budget with 10 days left.                    │
   │                             │  Limit daily spending to ₹200"                                    │
   ├─────────────────────────────┼───────────────────────────────────────────────────────────────────┤
   │ Recurring high expenses     │ "You spend ₹1,500/month on subscriptions.                        │
   │                             │  Review if all are needed"                                        │
   ├─────────────────────────────┼───────────────────────────────────────────────────────────────────┤
   │ Unusual transaction         │ "₹15,000 expense today is 5x your daily average.                 │
   │                             │  Was this planned?"                                               │
   └─────────────────────────────┴───────────────────────────────────────────────────────────────────┘

B) INVESTMENT SUGGESTIONS:
   ┌─────────────────────────────┬───────────────────────────────────────────────────────────────────┐

   │                             │  Consider investing in a SIP or FD"                               │
   ├─────────────────────────────┼───────────────────────────────────────────────────────────────────┤

C) SMART TIPS:
   - "50-30-20 Rule: Allocate 50% to needs, 30% to wants, 20% to savings"
   - "Your savings rate is 15% — try to push it to 20% this month"
   - "You have no income logged this month. Don't forget to add your salary!"


Implementation Approaches:
--------------------------

OPTION 1: Rule-Based Agent (Offline, No API needed) ✅ RECOMMENDED
   - Write logic that analyzes spending data locally
   - Uses if-else rules and formulas to generate tips
   - Pros: Free, works offline, fast, no privacy concerns
   - Example: If savings > 20% of income → suggest investment

OPTION 2: LLM-Powered Agent (API-based)
   - Send spending summary to an AI API (Gemini, OpenAI, etc.)
   - Get personalized, natural-language advice
   - Pros: More intelligent, conversational responses
   - Cons: Requires API key, internet, costs money, privacy concern


Where the Agent Would Appear in the App:
-----------------------------------------
1. New "Insights" Tab     — Dedicated tab with cards showing suggestions
2. Home Screen Banner     — Top tip of the day on the home page
3. Post-Transaction Nudge — After adding an expense, show a quick tip
4. Monthly Report Card    — End of month summary with agent's review


Recommendation:
---------------
Start with Option 1 (Rule-Based) — it's practical, works offline, and can
be built with existing SQLite data. An LLM layer can be added later for
more intelligent, conversational responses.
